# Skyzen

Skyzen is a lightweight HTTP framework built on top of the [`http-kit`](https://github.com/zen-rs/http-kit) stack.  

This repository contains the core framework crate, foundational traits in `skyzen-core`, and the optional Hyper backend in `skyzen-hyper`.
